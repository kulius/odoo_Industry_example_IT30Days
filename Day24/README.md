# Day 24 : 案例分享(7.3) 庫存與製造 - 從單純的製造開始

## 案例說明及適用場景
>- 自己購買原物料做組裝
>- 2~3個商品，透過包裝，形成庫存
>- 需扣料形成產品轉換

## 一.物料清單 (BOM)
>- 定義產生的產品及消耗的物料

## 二.查看BOM結構及成本
>- 定義的產品若為另一張BOM的消耗物料，可經結構成本表查看相關內容

## 三.製造訂單
