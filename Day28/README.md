## 功能說明
>- odoo在其運用中，訊息及討論，一直是個標準的內附功能
>- 在各單據中都可直接發送訊息給相關內部人員
>- 本次odoo15新線線上會議功能，功能與google meet一模一樣，而且不用註冊即可讓使用者透過瀏覽器參加會議
